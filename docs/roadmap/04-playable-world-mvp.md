# Playable World MVP

## What Exists Now

The repo now contains a playable terminal simulation where the player can interact with NPCs inside a running village world.

Current interaction surface:

- move between village locations
- inspect the current scene
- talk to NPCs
- ask NPCs for rumors
- buy and sell items with vendor NPCs
- use `next` while the world advances

## Runtime Features

- deterministic turn loop
- NPC hunger progression
- simple production jobs
- rumor propagation between nearby NPCs
- relationship-aware rumor sharing
- heuristic planner-backed NPC intents
- market price refresh from aggregate stock

## How To Run

```bash
python run_acidnet.py
```

Or, after editable install:

```bash
python -m pip install -e .
acidnet
```

## Useful Commands

- `look`
- `go tavern`
- `talk mara`
- `ask neri rumor`
- `trade mara buy bread 1`
- `status`
- `rumors`
- `next 3`

## Limits Of This MVP

- dialogue is persona-templated, not yet generated by a local fine-tuned model
- NPC planning is heuristic, not yet local-model inference
- memory is stored externally but not yet reflected into long-horizon summaries
- economy is intentionally simple and tuned for playability over realism
